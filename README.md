# Encrypt

加密/解密工具箱为用户提供了基于js的加密解密功能。

1. 加密
   1. URL编码
   2. Base64编码
   3. Md5哈希
   4. Sha1哈希
   5. Sha256哈希
   6. Sha512哈希
2. 解密
   1. URL解码
   2. Base64解码
